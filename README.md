## 💎 Design Philosophy

This repository isn't just a portfolio; it's a **Technical Audit Trail**. The UI is engineered to feel like a premium control tower, featuring:
- **Glassmorphic Interfaces:** High-focus, transparent UI elements.

## 📂 Project Architecture

The archive is strictly data-driven, separating the display logic from the technical content.

```text
├── src/
│   ├── components/       # Premium UI components (ProjectCard, Layout, etc.)
│   ├── data/             # The "Vault Data" (projects.json - 12+ production systems)
│   ├── views/            # High-fidelity page layouts (Home, Project Details)
│   ├── types.ts          # Strict documentation of the Project schema
│   └── style.css         # Custom global design tokens
├── public/               # Static assets & artifacts
└── package.json          # Dependency manifest
```
